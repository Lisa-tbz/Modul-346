# Teil A - Grundbegriffe erklären

### VPC
Die Virtual Private Cloud bezeichnet das eigene (Sub)netz innerhalb eines Anbieters, in unserem Fall AWS.

### Subnetz
Ein Subnetz ist eine weitere Fächerung innerhalb der VPC. Alle erstellten Server werden in Subnetzten platziert.

### Public IP
Die öffentliche IP ist innerhalb des globalen Internets und wird durch den ISP zugewiesen.

### Private IP
Die private IP ist im lokalen Netzwerk. Sie wird durch den Router als Gateway zwischen public und private IP vergeben. 
Sie ist da, um eine konsistenze Kommunikation innerhalb des eignen Netzes zu ermöglichen.

### Static IP
Die statische IP (egal ob private oder public) bleibt gleich, wöhrend dynamische IPs sich immer wieder abändern.
